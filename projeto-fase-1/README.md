Para executar este projeto:

1. Entre pasta projeto no terminal:
```
cd projeto
```

2. Rode npm install para instalar as dependências do projeto:
```
npm install
```

3. E em seguida, npm start, para iniciar a execução do projeto.
```
npm start
```

## Introdução

Fase 1 do projeto de Desenvolvimento Frontend no curso de Análise e Desenvolvimento de Sistemas na PUCRS

## Componentes

Os componentes estão no diretório `./src/components` e eles seguem as especificações do projeto, sendo eles:

- SeriesForm:
  - Descrição: este componente utiliza o form para adicionar séries a uma lista de séries que será posteriormente renderizada em outro componente.

- SeriesList:
  - Descrição: este componente renderiza a lista de Series que é criada e incrementada no componente SeriesForm, adicionando também botões para Editar e Excluir entradas na lista.

- Navbar:
  - Descrição: este componente utiliza Link do React Router para mostrar as páginas disponíveis em uma barra de navegação.

## Estilização

Os arquivos de estilização estão no diretório `./src/components/styles` e eles servem para dar estrutura ao projeto e deixá-lo mais fácil de entender e visualmente satisfatório
